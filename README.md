Welcome to <b>Dad Jokes</b>! <br/>
Generate random dad jokes using this gem.

## Installation

Add this line to your application's Gemfile:

``` ruby
gem 'dad_jokes'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dad_jokes
